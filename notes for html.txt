--The two important things to create the HTML pages?
---1. Elements / tags

---Element(s)/tag(s):
-----:- The Elements is used to represent the format of the particular content
---------There are Two Types of Elements
         1. Block
             <elementName>
             .......
             .......
             </elementName>
             ex. <html>......</html>, <p>...</p>, <body>...</body>,title,div,span,ect...,


         2. Inline:
            <elemenetName />
            ex., <br />, <hr />, <img />, <input />



        


--Attribute(s):
        :-The attribute are used to add the Charascteristics / Properties to the Elements
         1. Block
             <elementName attributeName="itsValue" attribute="itsValue">
             .......
             .......
             </elementName>
             ex. <html>......</html>, <p>...</p>, <body>...</body>,title,div,span,ect...,


         2. Inline:
